## 如何升级
因数据都是存储在浏览器本地，需重新导入数据才能使用新版本功能。
1. 导出数据：“我的”->“导出数据”。
2. 删除旧版本应用或书签。
3. 浏览器打开网址 record.sdgarden.top ,“分享”->“添加到主屏幕”。
4. 导入数据打开桌面的“作业积分”应用，点击：“我的”->“导入数据”。
